Henrick alpha (non linear weights) object.

Note

The provided alpha implements the alpha coefficients defined in Mapped weighted essentially non-oscillatory schemes: Achieving optimal order near critical points, Andrew K. Henrick, Tariq D. Aslam, Joseph M. Powers, JCP, 2005, vol. 207, pp. 542-567, doi:10.1016/j.jcp.2005.01.023
