Jiang-Shu alpha (non linear weights) object.

Note

The provided alpha implements the alpha coefficients defined in Efficient Implementation of Weighted ENO Schemes, Guang-Shan Jiang, Chi-Wang Shu, JCP, 1996, vol. 126, pp. 202–228, doi:10.1006/jcph.1996.0130.
