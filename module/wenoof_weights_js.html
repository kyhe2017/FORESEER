Jiang-Shu and Gerolymos-Senechal-Vallet weights.

Note

The provided WENO weights implements the weights defined in Efficient Implementation of Weighted ENO Schemes, Guang-Shan Jiang, Chi-Wang Shu, JCP, 1996, vol. 126, pp. 202–228, doi:10.1006/jcph.1996.0130 and Very-high-order weno schemes, G. A. Gerolymos, D. Senechal, I. Vallet, JCP, 2009, vol. 228, pp. 8481-8524, doi:10.1016/j.jcp.2009.07.039
